This app displays infinite postings fetched from [jsonplaceholder API](https://jsonplaceholder.typicode.com). 

The initial number of postings is 5. If a user scolls down, the app fetches 5 more postings. Searching posts that include specific keyword is possible. 




## Lessons Learned

- Fetching data with custom hook
- IntersectionObserver
- useRef, useCallback
- loading css
- useLocation()
